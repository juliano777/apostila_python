## float

Ponto flutuante; não tem precisão absoluta, sua precisão é relativa.  
Para uma maior precisão com números que tenham ponto flutuante, utilizar o módulo `decimal`.

```python
# Criação de um float:
f = float(3)
```

ou

```python
f = 3.0
```

```python
f  # Veririca o valor
```

```
3.0
```

Formas de se definir um float:

```python
x = 0.5000000000
```

ou

```python
x = 0.5
```

ou

```python
x = .5

x  # Exibe o valor
```

```
0.5
```

```python
type(x)  # Tipo
```

```
float
```

```python
x = 2.

x  # Verifica o valor
```

```
2.0
```

```python
# Que tipo resulta de da soma de um inteiro e um float?:
type(7 + 3.0)
```

```
float
```

```python
# Resultado:
7 + 3.0
```

```
10.0
```

```python
# Divisão:
3 / 2
```

ou

```python
3 / 2.0
```

ou

```python
3.0 / 2
```

ou

```python
3.0 / 2.0
```

```
1.5
```

```python
# Divisão Inteira:
3 // 2.0
```

```
1.0
```

```python
# Notação Científica:
1e+2
```

```
100.0
```

```python
# Notação científica com expoente negativo:
1e-3
```

```
0.001
```
