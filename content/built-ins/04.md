## len()

Função que retorna a quantidade de itens de um contêiner.

``` python
# Criação de um objeto contêiner e verificação da quantidade de elementos
foo = ('x', 'y', 'z', 123, 5.7)

# len(foo)
```

``` console
5
```

``` python
# Tamanho de uma string
len('Heavy Metal')
```

``` console
11    
```
