## Docstrings

São *strings* que vêm logo após a definição de uma função, de um método ou de
uma classe.  
É muito útil para fins de documentação.  
Para visualizar o conteúdo dessa string utiliza-se o atributo mágico `__doc__`
ou a função `help()`.

```python
# Criação de uma função
def foo():
    'Uma simples função'
```

```python
# Exibe a docstring da função
print(foo.__doc__)
```

```
Uma simples função
```

```python
# Criação de função
def bar():
    '''
    Mais outra
    função
    que não faz
    nada
    '''
```

```python
# Exibe a docstring da função
print(bar.__doc__)
```

```
Mais outra
função
que não faz 
nada
```

```python
# Criação de uma classe
class Foo(object):
    '''
    Uma classe
    de teste
    '''
```

```python
# Exibe a docstring da classe
print(Foo.__doc__)
```

```
Uma classe
de teste
```

```python
# Help da classe:
help(Foo)
```
