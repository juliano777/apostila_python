Breve Descrição

Operador

Breve Descrição

id(t1) 139965970848192

t3 = (1, 2, 3)

id(t3) 139965970847744

t1 is t2 True

t1 is t3 False

t1 == t3 True

t1 is not t3 True

# Operador pipe “|”: Ou Binário / Binary Or

‘’’

O operador pipe faz a lógica “or” binária, em português “ou”.

Dada a seguinte tabela da verdade em que;

0 = False 1 = True

15.8. Atribuição de Valores¶

# Atribuição Simples

foo = 0
bar = 'bla bla bla'
print(foo)

0

print(bar)

bla bla bla

# Atribuição Composta ou Atribuição por Tupla
x, y, z = (1, 2, 3)

print(x)

1

print(y)

2

print(z)

3

Invertendo valores:

x = 10
y = 20
x, y = y, x
print(x)

20

print(y)

10
