11. Lista¶

Uma lista é uma estrutura de dados similar ao tradicional array que tem em outras linguagens, porém seus elementos podem ser de tipos diferentes. Uma lista é também uma coleção de dados mutável, pois sua estrutura pode ser mudada.

Formas de criar uma lista vazia:

> l = []

ou

> l = list()

Formas de criar uma lista com elementos:

> l = list([1, 2, 3])

ou

> l = [1, 2, 3]

Declaração de uma nova lista:

> x = [‘a’, ‘b’, 3, 4.0]

Exibindo o terceiro elemento da lista:

> print(x[2])

3

Alterando o terceiro elemento da lista:

> x[2] = 30

Exibindo o terceiro elemento da lista:

> print(x[2])

30

Exibindo os elementos da lista:

> print(x)

[‘a’, ‘b’, 30, 4.0]

Existe “7” na lista?:

> 7 in x

False

Existe “3” na lista?:

> 3 in x

True

O método append adiciona um elemento ao final da lista:

> x.append(‘Uma string qualquer…’)

Exibindo os elementos da lista:

> print(x)

[‘a’, ‘b’, 30, 4.0, ‘Uma string qualquer…’]

O método extend faz com que os elementos de outra lista sejam adicionadas a uma lista atual:

> foo = [‘a’, ‘b’, ‘c’] # Definição da primeira lista

> bar = [1, 2] # Definição da segunda lista

> foo.extend(bar) # Extendendo a primeira lista com os elementos da segunda

> print(foo) # Exibindo a lista extendida

[‘a’, ‘b’, ‘c’, 1, 2]

O operador “+” sendo utilizado para criar uma nova lista a partir de outra somada à outra:

> y = x + [‘abobrinha’, ‘7’, {‘chave’: ‘valor’}, (‘SP’, ‘MG’, ‘PR’, ‘RO’)]

Exibindo a nova lista:

> y

[‘a’,

‘b’, 30, 4.0, ‘Uma string qualquer…’, ‘abobrinha’, ‘7’, {‘chave’: ‘valor’}, (‘SP’, ‘MG’, ‘PR’, ‘RO’)]

Utilizando “+=” como um atalho para o método extend e exibindo seu novo conteúdo:

> x += [‘xyz’]

> x

[‘a’, ‘b’, 30, 4.0, ‘Uma string qualquer…’, ‘xyz’]

Removendo um elemento da lista e exibindo seu novo conteúdo:

> x.remove(‘xyz’)

> x

[‘a’, ‘b’, 30, 4.0, ‘Uma string qualquer…’]

Método extend com uma lista como parâmetro:

> x.extend([‘xyz’])

> x

[‘a’, ‘b’, 30, 4.0, ‘Uma string qualquer…’, ‘xyz’]

Método extend com uma string como parâmetro:

x.extend(‘xyz’)

> x

[‘a’, ‘b’, 30, 4.0, ‘Uma string qualquer…’, ‘xyz’, ‘x’, ‘y’, ‘z’]

Ao utilizarmos uma string como parâmetro do método extend, a string foi convertida em lista de forma a transformar cada caractere em elemento de uma lista.

Método append:

> x.append(‘xyz’)

> x

[‘a’, ‘b’, 30, 4.0, ‘Uma string qualquer…’, ‘xyz’, ‘x’, ‘y’, ‘z’, ‘xyz’]

Aqui podemos ver claramente a diferença entre os métodos extend e append, que pode causar uma certa confusão inicial para quem está iniciando em Python. Nota-se que o método append adicionou a string inteira como um novo elemento da lista.

Extendendo a lista com o operador “+=”:

> x += ‘String’

> x

[‘a’,

‘b’, 30, 4.0, ‘Uma string qualquer…’, ‘xyz’, ‘x’, ‘y’, ‘z’, ‘xyz’, ‘S’, ‘t’, ‘r’, ‘i’, ‘n’, ‘g’]

Como o operador “+=” ser um atalho para o método extend ele transformou a string numa lista, uma lista cujos elementos são os caracteres da string.

Tentativa de utilizar o operador “+” em uma lista:

> x + ‘bla bla bla’

TypeError: can only concatenate list (not “str”) to list

Não se pode concatenar uma lista com uma string.

Criação de uma nova lista a partir de uma string:

> z = list(‘Hobbit’)

> z

[‘H’, ‘o’, ‘b’, ‘b’, ‘i’, ‘t’]

Novamente o método append:

> z.append(‘Hobbit’)

> z

[‘H’, ‘o’, ‘b’, ‘b’, ‘i’, ‘t’, ‘Hobbit’]

Qual o tamanho (quantos elementos) da lista?

> len(z)

7

Tamanho 7, posições variam de 0 a 6.

Método insert utilizando a priemira posição (0):

> z.insert(0, ‘Gandalf’)

> z

[‘Gandalf’, ‘H’, ‘o’, ‘b’, ‘b’, ‘i’, ‘t’, ‘Hobbit’]

Pode-se também verificar o tamanho de uma lista com o dunder len:

> z.__len__()

8

Dado que a lista tem 8 (oito) elementos, inserir num novo elemento na nona (8) posição:

> z.insert(8, ‘Bilbo’)

> z

[‘Gandalf’, ‘H’, ‘o’, ‘b’, ‘b’, ‘i’, ‘t’, ‘Hobbit’, ‘Bilbo’]

O efeito foi o mesmo que utilizar o método append.

O método pop sem parâmetros retorna o último elemento e o apaga da lista:

> z.pop()

‘Bilbo’

> z

[‘Gandalf’, ‘H’, ‘o’, ‘b’, ‘b’, ‘i’, ‘t’, ‘Hobbit’]

Como o método pop retorna o último elemento, o mesmo pode ser utilizado para atribuição:

> livro = (‘O {}’.format(z.pop()))

> livro

‘O Hobbit’

Retornando e apagando o sexto elemento:

> z.pop(5)

‘i’

Método sort, organiza a lista com seus elementos pela ordem alfabética:

> z.sort()

> z

[‘Gandalf’, ‘H’, ‘b’, ‘b’, ‘i’, ‘o’, ‘t’]

O método reverse pega a atual posição dos elementos e reconstrói a lista na ordem reversa:

> z.reverse()

> z

[‘t’, ‘o’, ‘b’, ‘b’, ‘H’, ‘Gandalf’]

A função sorted não altera a lista, apenas retorna o conteúdo pela ordem alfabética:

> sorted(z)

[‘Gandalf’, ‘H’, ‘b’, ‘b’, ‘o’, ‘t’]

> z

[‘t’, ‘o’, ‘i’, ‘b’, ‘b’, ‘H’, ‘Gandalf’]

A função sorted também pode somente retornar o reverso de uma lista:

> sorted([1, 2, 3], reverse=True)

[3, 2, 1]

Criação de uma lista a partir do retorno de sorted:

> w = sorted(z)

> w

[‘Gandalf’, ‘H’, ‘b’, ‘b’, ‘o’, ‘t’]

> z

[‘t’, ‘o’, ‘b’, ‘b’, ‘H’, ‘Gandalf’]

A função reversed() sempre retorna um iterador:

> reversed(z)

<listreverseiterator object at 0x1653c10>

Convertendo para lista o iterador gerado pela função reversed:

> z = list(reversed(z))

> z

[‘Gandalf’, ‘H’, ‘b’, ‘b’, ‘o’, ‘t’]

Função sorted transforma a string numa lista e organiza por ordem alfabética:

> sorted(‘aAcb’)

[‘A’, ‘a’, ‘b’, ‘c’]

Função sorted transforma a string numa lista e organiza por ordem alfabética:

> sorted(‘aAcb’, reverse=True)

[‘c’, ‘b’, ‘a’, ‘A’]

Função reversed transforma a string em uma lista com seus caracteres em ordem reversa:

> list(reversed(‘aAcb’))

[‘b’, ‘c’, ‘A’, ‘a’]

Definição de lista com 3 (três) elementos:

> x = [1, 2, 3]

A partir da lista “x”, atribuir respectivamente seus elementos como valores para as variáveis à esquerdae exibir seus valores:

> a, b, c = x

> print(a)

1

> print(b)

2

> print(c)

3

E se a quantidade de variáveis que receberão os valores forem em menor número que a quantidade de elementos da lista?:

> y, z, = x

ValueError: too many values to unpack (expected 2)

Lista com três elementos não pode fazer atribuição respectiva a apenas duas variáveis.

Utilização do caractere underscore como solução:

> y, z, _ = x

> print(y)

1

> print(z)

2
