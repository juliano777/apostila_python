22. Loops - Laços de Repetição¶

22.1. while¶

Executa em laço (loop) enquanto a condição for verdadeira.

i = 0

while i < 5:

print(i) i += 1

0 1 2 3 4

i = 0

22.1.1. O else no loop while¶

Opcionalmente, pode-se adicionar um else ao while em Python. A idéia é que se caso o loop seja executado sem interrupção, um break, por exemplo, o que estiver dentro do bloco else será executado.

while i < 5:

print(i) i += 1

else:

print(‘Fim’)

0 1 2 3 4 Fim

i = 0

while i <= 10:

if i == 5: break print(i) i += 1

else:

print(‘Fim’)

0 1 2 3 4

i = 0

while i <= 10:
if (i % 2 == 0):

i += 1 continue

print(i) i += 1

else:

print(‘Fim’)
