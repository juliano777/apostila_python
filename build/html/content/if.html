22. if / elif / else¶

O comando if, que inglês significa “se” indica uma condição.

# Criação de dois objetos int:
x = 7
y = 5

# Bloco if:
if x > y:
    print('X é maior')

X é maior

# Bloco if;
if x < y:
    print('X é maior')

# Objetos booleanos:
foo = True
bar = False

# Bloco if:
if foo:
    print('foo é verdadeiro!')

foo é verdadeiro!

# Bloco if:
if not bar:
    print('bar é falso!')

bar é falso!

# Objeto string:
texto = 'Python e PostgreSQL: Poder!'

# Bloco if:
if texto:
    print('A string NÃO é vazia!')

A string NÃO é vazia!

# String vazia:
texto = ''

# Bloco if:
if not texto:
    print('A string é vazia!')

A string é vazia!

# Objetos int:
x = 1
y = 2

# Bloco if:
if x > y:
    print('X é maior')
else:
    print('Y é maior')

Y é maior

# Objetos int:
y = 1
x = 1

# Bloco if:
if x > y:
    print('X é maior')
elif x < y:
    print('Y é maior')
else:
    print('Valores iguais')

Valores iguais

# Objeto int:
x = 10

# Bloco if:
if (x > 5):
    y = 3
else:
    y = 0

22.1. if Ternário¶

x = 10  # Variável int

# Atribuição de valor condicional:
y = (50 if (x > 5) else 40)

# Exibe o valor de "y":
print(y)

50

# Variável float para receber a nota:
nota = float(input('Digite a nota do aluno: '))

Digite a nota do aluno: 8

# Atribuição condicional:
estado = 'aprovado' if nota >= 7 else 'reprovado'

# Exibe a mensagem:
print('Aluno {}!'.format(estado))

Aluno aprovado!

# Variável int:
num = int(input('Digite um número: '))

Digite um número: -2

# Atribuição condicional:
sinal = 'positivo' if num > 0 else 'negativo' if num < 0 else 'zero'

# Exibe mensagem:
print('O número é {}'.format(sinal))

O número é negativo
